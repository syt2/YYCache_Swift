## Installation

YYCache_Swift is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'YYCache_Swift', :git => 'https://github.com/syt2/YYCache_Swift.git'
```
